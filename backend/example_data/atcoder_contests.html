CCC usually holds all of their practice contests when it's not ICPC season on Codeforces. However, did you know that there are other websites like AtCoder that has problems? AtCoder holds four different types of contests
  • ABC: AtCoder Beginner Contests
  • ARC: AtCoder Regular Contests
  • AGC: AtCoder Grand Contests
  • AHC: AtCoder Heuristic Contests
You are given three of the names of these contests. Output the one that is missing from the list.

Input

You will be given three strings separated by newlines $S_1,S_2,S_3$. It is guaranteed that $S_1,S_2,S_3$ is one of ABC, ARC, AGC, AHC. It is also guaranteed that they are pairwise distinct.

Output

Output the missing contest from the list.

Examples

Input 
ARC
AGC
AHC
Output 
ABC
Input 
AGC
ABC
ARC
Output 
AHC
