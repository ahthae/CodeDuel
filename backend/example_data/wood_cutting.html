You are given a long piece of timber that is of length $L$ meters. For each $x={1,2,...,L-1}$ there is a mark that is placed at $x$ meters from the left end of the piece. You are given $Q$ queries and the $i$-th query is represented as a pair of numbers $(c_i,x_i)$.
  • If $c_i = 1$, you cut the timber just to the right of the mark placed at position $x$. This will make the piece that you cut into two different pieces.
  • If $c_i=2$, you choose the piece that has a mark placed at position $x$ and print its length.
It is guaranteed that there will be no cut ade at the mark $x_i$ when the query is to be processed. The cuts are permanent and last after the query they were made.

Input

The first line contains $L$ $(1 \le L \le 10^9)$ and $Q$ $(1 \le Q \le 2 \times 10^5)$, the initial length of the timer and the number of queries you will be asked. The next $Q$ lines contain a pair of integers $(c_i,x_i)$ $(1 \le x_i \le L-1)$, the type of the query and the mark that you need to process.

Output

You should have the same number of lines as the number of type $2$ queries.
