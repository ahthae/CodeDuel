Recall that a permutation of length $N$ is a list of numbers where the numbers $1$ through $N$ occur exactly once. Given a permutation $P=(p_1,...,p_N)$ output a permutation $Q=(q_1,...,q_N)$ such that for every $i={1,..,N}$, the $p_i$-th element in $Q$ is $i$. It can be proved that there exist a unique Q that satisfies the condition.

Input

The first line will contain $N$ $(1 \le N \le 2 \times 10^5)$, which is the number of elements in $P$. The next line will contain the permutation $P$ which is of length $N$.

Output

Output the permutation $Q$ in one line.

Examples

Input 
3
2 3 1
Output 
3 1 2
Input 
3
1 2 3
Output 
1 2 3
Input 
5
5 3 2 4 1
Output 
5 3 2 4 1
